Nabman

A QBasic game I wrote in 1998

I made Nabman when I was about 14 years old. At the time I'd had no formal programming training of any kind.

It was loosely inspired by Pacman (at least at first) and lacked support for multiple ghost characters, internal walls, and a timer to control the pace of the game. As a result the ghost moves around the level in random directions as quickly as your computer can manage. This made the game extremely difficult to play so the player starts off with 10 lives, with one being deducted each time the ghost and player occupy the same tile.

The objective of the game is to get the smiley face from the top left of the screen to the bottom right, press space bar, and then return to the top left and press space bar again. After hitting space bar in the bottom right corner, the player is rewarded with 10 additional lives to help them complete the return journey. If the player succeeds, they are asked to enter their name in the 'High Scores' list. However if they die, a skull-and-bones image scrolls across the screen to the tune of Beethoven's Symphony No. 5.

As you can tell from the 'Credits' screen, all graphic and audio artwork was created by me :)

The source code has been uploaded for nostalgic and reference purposes only - it is no longer an active project.

Dependencies

  • QBASIC.EXE

    Definitely included with all versions of Microsoft Windows until (and including) Windows 98 ME. If you don't have it, you can download it from the link above as part of the Windows 95 Extras CD-ROM.

  • DOSBox

    A DOS emulator - useful if you do not run Windows

Install

Copy all the files in the repository to "C:\nabman\". If you are using DOSBox, make sure you mount your filesystem such that "C:\nabman\" contains the source files. Start up QBASIC.EXE, load MAIN.BAS and click [Run -> Start]
