Nabman by nelisa

A game I wrote in QBasic in 1998 :) Wow, nostalgia...

I wrote Nabman when I was about 14 years old. At the time I'd had no formal programming training of any kind, which will probably becomes clear if you manage to get the game to run! It was inspired by Pacman and Nibbles - another QBasic game quite similar to the Snake game that shipped with many Nokia phones around that time. Needless to say it resembles neither, partly because I never got round to finishing it but mostly because it runs at a ridiculously fast pace (I didn't know how to implement timers properly!).

Dependencies

QBASIC.EXE

Install

Copy all the files in the repository to "C:\nabman\". To start the game, run the MAIN.BAS file.
